Projects

  • Python Validation Software

    When tasked with improving the validation process at my employer. I took the on the challenge by breaking down the situation into bite size pieces. This particular task required me to control the temperature of the chamber, the pressure of the chamber, and the voltage on the device. Doing so taught me the various techniques for communicating with different pieces of validation hardware. For example the multimeter, power supply, pressure controller and temperature chamber. Each of these devices required different commands to make sure they were controlled properly and at the approriate time. I was able to break down the testing into simple steps.

    1. Set temperature
    2. Verify temperature
    3. Measure temperature
    4. Set pressure
    5. Verify pressure
    6. Measure pressure
    7. Repeat for all temperatures
    Using simple input questions I am able to use this validation software for any validation test plan that uses this equipment. This software has reduced the validation test time from 2 weeks down to 2 days, increasing the efficiency by 700%

  • Improve Company Design for Manufacturing Process

    While beginning my career at my employer, I quickly notices the lack of structure around the process for designing new PCBA's. This process did not take in to account any of the standards located in the IPC standards. These guidelines are used to make PCBA's manufacturable at multiple contract manufacturers. This was leading to issues when changing suppliers for PCBAs when one supplier did not have the capabilities that the previous supplier did. By standardizing the design rules to be compatible with the IPC standards our PCB designs quickly became easier to manufacure, and cheaper.
